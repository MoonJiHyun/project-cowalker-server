# 질문

## 질문 조회

| 메소드 | 경로                    | 짧은 설명            |
| ------ | ----------------------- | -------------------- |
| GET    | /question/{recruit_idx} | 참여하기의 질문 조회 |

### 요청

```json
/question/5b3de49ed711536c203dbe72
```

#### 조회 성공

```json
{
    "message": "success",
    "result": [
        "어디사세요?",
        "질문수정3"
    ]
}
```

#### 조회 실패

```json
{
    "message": "database failure"
}
```

#### 조회 실패 ( DB에 없는 recruit_idx를 줄 경우)

```json
{
    "message" : "Bad Request"
}
```
