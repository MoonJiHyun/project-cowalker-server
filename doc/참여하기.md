# 지원서

## 지원서 작성

| 메소드 | 경로   | 짧은 설명 |
| ------ | ------ | --------- |
| POST   | /apply | 지원서 작성  |

### 요청 헤더

```
Content-Type: application/json
authorization : 로그인 토큰 값
```

### 요청 바디

* 안드로이드,웹은 portfolio_url에 파일 첨부, ios는 포트폴리오 링크

```json
{
    "introduce" : "소개",
    "portfolio_url" : "포트폴리오 주소 혹은 포트폴리오 업로드",
    "phone" : "연락처",
    "recruit_idx" : "모집 공고 id",
    "project_idx" : "프로젝트 id",
    "position" : "모집 역할",
    "answers" : ["1번째 질문의 대답", "2번째 질문의 대답"]
}
```

### 요청 바디 예시

* 안드로이드,웹은 portfolio_url에 파일 첨부, ios는 포트폴리오 링크

```json
{
    "introduce" : "안녕하세요.저는 문지현입니다.",
    "portfolio_url" : "https://project-cowalker.s3.ap-northeast-2.amazonaws.com/1530643487481.txt",
    "phone" : "010-1234-5678",
    "recruit_idx" : "11",
    "answers" : ["잘 할 수 있습니다.", "뽑아주세요."]
}
```

### 응답 바디

#### 생성 성공

```json
{
    "message": "success"
}
```

#### 생성 실패

```json
{
    "message": "database failure"
}
```

#### token 없을 때

```json
{
    "message": "access denied"
}
```

## 개설자 지원서 조회

| 메소드 | 경로                                  | 짧은 설명                                   |
| ------ | ------------------------------------- | ------------------------------------------- |
| GET    | /:apply_idx/:applicant_idx/join/:join | 참여 요청에 대한 개설자의 참여 수락 or 거절 |

### 요청 헤더

```
Content-Type: application/json
authorization : 로그인 토큰 값
```

### 요청 파라미터

```
apply_idx : 지원 고유 idx
applicant_idx : 지원자 idx 
join : 참여 여부 (수락 : 1, 거절 : 2 | default : 0)
```

#### 조회 성공


```json
    {
        "apply_idx": "5b3d2b1de440364e0ec136a9",
        "introduce": "안녕하세요.저는 문지현입니다.",
        "portfolio_url": "https://project-cowalker.s3.ap-northeast-2.amazonaws.com/1530643487481.txt",
        "phone" : "010-1234-5678",
        "recruit_idx": "11",
        "applicant_idx": "45",
        "recruit_at": "2018-07-04T20:16:29.877Z",
        "answers": ["잘 할 수 있습니다.", "뽑아주세요."]
    }
```

#### 조회 실패

```json
{
    "message": "database failure"
}
```

#### token 없을 때

```json
{
    "message": "access denied"
}
```

#### 권한이 없을 때

```json
{
    "message": "fail (no rights)"
}
```
