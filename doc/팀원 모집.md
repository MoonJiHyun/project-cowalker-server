# 팀원 모집

## 모집하기 

메소드 | 경로 | 짧은 설명
--- | --- | ---
POST | /recruit | 팀원 모집 

### 요청 헤더
~~~
authorization : token
~~~
### 요청 바디
```json
{
	"position" : "PM",
	"start_date" : "2018-07-03",
	"end_date" : "2018-07-20",
    "number" : 3,
    "task": "활동",
    "activitiy" : "활동 시간 및 기간",
    "reward" : "혜택 및 보상",
    "area" : "활동 위치(상세주소)",
    "ability" : "필요 역량과 기술",
	"career":"요구 경력 및 경험",
    "preference":"우대사항",
    "comment" : "코멘트"
}
```
```json
* start_date, end_date : String (-넣어서 보내주세요)
```

### 응답 바디

#### 모집 실패
```json
{
    "message": "database failure"
}
```
#### 모집 성공
```json
{
    "message": "success"
}
```

## 모집하기 조회

| 메소드 | 경로                                         | 짧은 설명           |
| ------ | -------------------------------------------- | ------------------- |
| GET    | /project/{project_idx}/recruit/{recruit_idx} | 모집 공고 세부 조회 |

### 요청 헤더

```
authorization : token
```

### 요청

```json
/project/5b3dd2387172d402215033d2/recruit/5b3ecc11ca5c3444e4f802f1
```

#### 조회 성공

```json
{
    "message": "success",
    "result": [
        {
            "start_date": "2018-07-03T00:00:00.000Z",
            "end_date": "2018-07-20T00:00:00.000Z",
            "position": "서버 ",
            "number": 4,
            "task": "활동",
            "activity": "활동 기간입니다",
            "reward": "보상수정",
            "area": "위치
            "필요역량 수정본\n",
            "career": "경력수",
            "preference": "노원 주민 우대",
            "comment": "서버 잘하시는 분!! 찾아요!! "
        }
    ],
    "btnResult": "참여하기"
}
```

```json
* btnResult  -> 총 4가지의 value값을 가진다. 

 - btnResult : "개발자 " 
 - btnResult : "참여하기"
 - btnResult : "참여대기"
 - btnResult : "참여완료"
```

#### 조회 실패

```json
{
    "message": "database failure"
}
```
